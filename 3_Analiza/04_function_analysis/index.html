1) \(y = x^2 \ln x\)

2) \(y = (\ln x)^3 - 3 \ln x\)

3) \(y = \frac{1}{\ln x}\)

4) \(y = \ln x + \frac{1}{\ln x}\)

5) \(y = \frac{1}{2} x^2 \ln \frac{1}{2} x\)

7) \(y = \frac{1+\ln x}{x}\)

8) \(y = \ln(x^2-1) + \frac{1}{x^2-1}\)

10) \(y = \frac{\ln x}{\sqrt{x}}\)

11) \(y = \ln \left(e + \frac{1}{x}\right)\)

16) \(y = \frac{x}{e^{x}-1}\)

17) \(y = e^{\frac{1}{x^2-1}}\)

18) \(y = e^{\frac{1}{x}}(x+1)\)

19) \(y = e^{\frac{x^2}{x^2-1}}\)

20) \(y = e^{-x^2}\)

21) \(y = e^{-x^2+8x-14}\)

22) \(y = xe^{-\frac{1}{2}x^2}\)

23) \(y = x^2 e^{1/x}\)

24) \(y = x^3 e^{-4x}\)

25) \(y = \left(a+\frac{x^2}{a}\right) e^{x/a}\)

27) \(y = e^{\text{tg } x}\)

28) \(y = e^{\text{arctg } x}\)
